# CSV Utility Scripts (Windows Batch)

CSVファイルの編集や、Excelでの文字化け防止を目的とした、Windows向けのバッチスクリプト集です。
インストール不要で、CSVファイルをバッチファイルにドラッグ＆ドロップするだけでご利用いただけます。

## 収録ファイル一覧

| ファイル名 | 役割 | 出力先 |
| :--- | :--- | :--- |
| **`csvQuoted.bat`** | 全ての項目（フィールド）を `"` で囲む | 元ファイルと同一フォルダに `[元ファイル名]_quoted.csv` を作成 |
| **`csvToUtf8WithBOM.bat`** | Excel用に文字コードを「BOM付きUTF-8」に変換して開く | 元ファイルと同一フォルダに `[元ファイル名]_bom.csv` を作成（※） |

※ すでに「BOM付きUTF-8」の場合は新規ファイルを作成せず、元のファイルをそのまま開きます。

---

## 使い方

1. 処理したいCSVファイルを、対象のバッチファイルに**ドラッグ＆ドロップ**します。
2. 自動的にPowerShellが起動し、処理を実行します。処理終了後に何かキーを押すと画面が閉じます。

---

## 各ツールの詳細動作

### 1. csvQuoted.bat (ダブルクォーテーション付与)
CSVデータの各列をダブルクォーテーション `"` で囲み直したファイルを作成します。

* **処理内容**:
  * UTF-8またはShift-JISで書かれたCSVファイルを自動判別して読み込みます。
  * すべての項目を `"` で囲みます。データ内に `"` が含まれている場合は `""` に適切にエスケープ処理を施します。
  * 出力ファイルの文字コードは「BOM付きUTF-8」となります。
* **主な用途**:
  * システム連携やデータベースへのインポート時に、項目を `"` で囲む必要がある場合。

### 2. csvToUtf8WithBOM.bat (文字化け対策・Excel起動)
CSVファイルの文字化けを防ぐために「BOM付きUTF-8」に文字コードを変換し、自動的にExcelで開きます。

* **処理内容**:
  * 入力されたCSVファイルの文字コードを自動解析します（BOM付きUTF-8 / BOMなしUTF-8 / Shift-JISなどを自動識別）。
  * すでに「BOM付きUTF-8」であれば、変換を行わずに直接Excelでファイルを開きます。
  * 「BOMなしUTF-8」や「Shift-JIS」の場合は、文字化けしにくい「BOM付きUTF-8」に変換した新しいファイル（末尾に `_bom`）を作成し、そちらをExcelで開きます。
* **主な用途**:
  * 他システムからダウンロードしたCSVファイルやUTF-8形式のファイルを、Excelで文字化けさせずに閲覧・編集したい場合。

## 動作環境・前提条件

* **OS**: Windows 10 / Windows 11
* **実行環境**: Windows PowerShell（OS標準搭載のもので動作します）
* **対象ファイル**: 拡張子が `.csv` のファイル（ドラッグ＆ドロップされたファイルがCSV以外の場合は、処理をスキップします）

---

## 安全性とプライバシーについて

* **完全ローカル処理**: 
  インターネットを介したデータの送信、外部サーバーへの通信処理は一切含んでいません。個人情報や機密データを含むCSVファイルであっても安全に使用できます。
* **オリジナルデータの保護**:
  元のCSVファイルを直接上書きする仕様ではなく、別名で新しいファイルを出力するため、元データが誤って破損・消失することはありません。
