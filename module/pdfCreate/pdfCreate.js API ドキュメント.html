ES Module

pdfCreate.js

HTML構造を整形し、しおり・ハイパーリンク・カスタムフォントを維持したまま、高精度なベクトルPDFに変換・出力するクライアントサイド用モジュールです。

注意事項 & 参照ライブラリ

導入および運用上の注意

  • フォントの読み込みは非同期処理です。PDFを出力する前に確実に読み込みが完了している必要があります。
  • 自分でフォントファイルを用意する場合は、pdfCreate.jsを使用するhtmlファイルの相対パスをfontUrlsのurlに指定してください。フォントファイルが見つからない場合は、標準フォント（Helvetica）で代用されます。
  • PDFの寸法は、指定した最初のページのサイズに基づいて自動判定されます。CSSで対象要素の幅・高さを適切に調整してください。
  • 変換時には、対象ページ要素に一時的に .pdf-print-active クラスが付与されます。レイアウト調整用CSSに活用してください。
  • 非印刷要素には .no-print クラスをあらかじめ付与してください。
  • デバッグモードを有効にすることで、処理のパフォーマンス計測やPDF出力前にプロンプトを一時停止させることが可能です。
ベース描画エンジン dompdf.js 結合・アノテーション注入 pdf-lib しおり (アウトライン) 構成 outline-pdf

動作要件・前提条件

本モジュールは、一部のライブラリがグローバルスコープ（`window` オブジェクト）に存在することを前提としています。モジュールを読み込む前に、HTML側で以下のスクリプトをロードしてください。

<!-- PDF描画コアエンジン -->
<!-- 使用するdompdf.min.jsはこちら -->
<script src="./dompdf.min.js"></script>

<!-- PDF結合・アノテーション操作ライブラリ -->
<script src="https://unpkg.com/pdf-lib/dist/pdf-lib.min.js"></script>

クイックスタート 

import { loadingFonts, exportPDF } from './pdfCreate.js';

// 1. フォントファイルの非同期読み込み
await loadingFonts({
    fontUrls: [
       { family: 'Zen Maru Gothic', url: './fonts/ZenMaruGothic-Regular.ttf', isDefault: true },
       { family: 'Zen Maru Gothic 700', url: './fonts/ZenMaruGothic-Bold.ttf', isDefault: false }
    ],
    msgElId: 'font-status-msg'
});

// 2. PDFの出力処理
await exportPDF({
   debug: false,
   overlayFont: 'Zen Maru Gothic 700',
   pagesSelector: '.a3-page',
   outlineStr: '1||Section 1\n1|-|Subsection 1.1\n2||Section 2',
   pdfFileName: 'output'
});

推奨CSSスタイル

出力時の一時的なスタイル制御を可能にするために、以下のスタイル定義を推奨します。

/* PDF出力時に適用される標準的なスタイル設定 */
.pdf-print-active {
   background-color: #ffffff !important;
   margin: 0 !important;
   box-shadow: none !important;
   border-radius: 0 !important;
}

/* 印刷対象外（ヘッダーやボタンなど）を隠す指定 */
.pdf-print-active .no-print {
   display: none !important;
}

プロパティ 型 説明
family string 登録するフォントファミリー名
url string フォントファイル（.ttfなど）の相対または絶対パス
isDefault boolean デフォルトフォントとして扱うかどうかのフラグ
@typedef

fontConfig

変換処理で用いられる、Base64形式にエンコード済みのフォント設定オブジェクトです。

プロパティ 型 デフォルト値 説明
fontFamily string - 登録用のフォントファミリー名
fontBase64 string - Base64に変換されたフォント文字列
fontUrl string '' オプションのURL記述子
fontWeight number 400 フォントの太さ（400は通常ウェイト）
fontStyle string 'normal' フォントスタイル（通常・斜体など）
isDefault boolean - デフォルトに指定するかどうかの真偽値
@typedef

debugInfo

debugを有効にした際の実行速度や、段階的なプロセスのミリ秒記録を蓄積するオブジェクトです。

オブジェクトパス プロパティ 型 説明
debugInfo.dompdf start / end number dompdf.jsによるHTMLレンダリングの開始・終了時刻
debugInfo.pdfLib start / end number pdf-lib結合、アノテーションインジェクションの開始・終了時刻
debugInfo.outlinePdf start / end number outline-pdfによる目次（アウトライン）処理の開始・終了時刻

公開メソッド (Public)

Public Async

loadingFonts()

渡されたフォントURL群を並行して非同期にダウンロードし、Base64形式に変換した上でモジュールスコープの変数へ格納します。 フォントの処理順序は、渡された配列順に完全固定されます。

パラメーター
引数名 型 デフォルト 説明
config Object {} 設定オブジェクト。
↳ fontUrls fontLoadingInfo[] - フォント情報配列。
↳ msgElId string '' 読み込み完了またはエラー時のメッセージを表示するためのHTML要素のID（任意）。
戻り値

Promise<void>

エラーハンドリング仕様

いずれかのフォントが取得不可能な場合は、例外を受け取ってフォールバックメッセージ（Helveticaによる代用）を表示し、安全に終了します。

Public Async

exportPDF()

ターゲットDOMの座標構造を取得・変形し、オーバーレイ表示で進捗を示しつつ、1ページごとにdompdf.jsでベクターPDFへと変換処理をかけます。 最終的にpdf-libおよびoutline-pdfを統合的に用い、しおりとしおり内ハイパーリンク付きの結合PDFファイルを生成し、自動ダウンロードをトリガーします。

パラメーター
引数名 型 デフォルト 説明
config Object {} 設定用オブジェクト。
↳ debug boolean false デバッグログの出力・処理時間測定。および出力確認用ボタンを表示。
↳ overlayFont string '' 読み込み中（進捗画面）で使用するフォントの名称。
↳ pagesSelector string 必須 印刷対象とする各ページコンポーネントのセレクタ名。
↳ outlineStr string '' 目次を構成するためのフォーマット済みアウトライン文字列。
↳ pdfFileName string 必須 ダウンロードされるファイルの名称（拡張子不要）。
戻り値

Promise<void>

injectAnnotations()

DOM上のアンカータグ <a href="..."> の実座標と、生成されたベクターPDF上の座標系とをマッピング（スケーリングと原点変換）し、 該当のPDFオブジェクト位置にインタラクティブなハイパーリンク（ベクターアノテーション）を動的に注入します。

座標計算ロジック

HTMLとPDFでは縦方向の原点が逆転している（HTMLは左上、PDFは左下）ため、以下の公式が適用されています。

scale = pdfPageWidth / parentRect.width;
x1 = (rect.left - parentRect.left) * scale;
y1 = pdfPageHeight - (rect.bottom - parentRect.top) * scale; // Y軸反転
パラメーター
引数 型 説明
pdfDoc PDFLib.PDFDocument PDFLibのドキュメントインスタンス
pages PDFLib.PDFPage[] 各ページのインスタンス配列
originalPages Object[] ソース元となるDOMツリーのエレメント群配列
Private

formatFontName()

フォント名をパースする際、CSSで予約された総称フォント名（sans-serifなど）を除くフォント名について、シングルクォーテーションで安全にラップする文字列処理を行います。

パラメーター

戻り値

string - フォーマットが標準化されたCSS用フォント定義文字列
